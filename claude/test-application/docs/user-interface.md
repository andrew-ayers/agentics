## user-interface.md

This file provides guidance to Claude Code (claude.ai/code) for the User Interface of the application.

## User Interface 

1. The application will be presented as a **Single Page (SPA), Progressive Web Application (PWA)**.
2. A manifest file for the PWA implementation is needed.

## User Interface Elements

1. **Title**: Read `docs/elements/title-element.md`
2. **Visual Representation**: Read `docs/elements/visual-rep-element.md`
3. **Numeric Input/Output Field**: Read `docs/elements/numeric-field-element.md`
4. **Reset Button**: Read `docs/elements/reset-button-element.md`

## User Interface Elements Layout

**Application Title**: Resistor Calculator
**Element Layout**: Stacked vertically and centered horizontally
**Order**: Title, Visual Representation, Numeric Input/Output Field, Reset Button
**Default State**: The **Visual Representation** has no colors selected, and the **Numeric Input/Output Field** element is empty/blank.

See also `docs/ui-layout.md` for an ASCII representation of the layout.
