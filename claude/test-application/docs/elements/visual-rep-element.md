## visual-rep-element.md

This file provides guidance to Claude Code (claude.ai/code) for the Visual Representation element (of a resistor) of the user interface.

## Visual Representation

**Order**: 1
**Type**: SVG (Scalable Vector Graphics)

## Details

1. **Shape**: Rectangular, long-axis horizontal
2. **Color**: Filled, beige or putty color
3. **Bands**: Four color bands representing the resistance value
    i. **First three bands**: Evenly spaced from the left edge of the shape
   ii. **Fourth band**: Displayed toward the right edge of the shape, spaced away from the first three bands
