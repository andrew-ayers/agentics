## Notes

1. The ASCII Representation of Layout should serve for layout guidance only.
2. The resistance color bands are denoted as 1, 2, 3 & 4 on the ASCII layout of the **Visual Representation**.
3. "Value in Ohms" and the surrounding "box" represents the **Numeric Input/Output Field**.
4. "Reset" and the surrounding "box" represents the **Reset Button**.
