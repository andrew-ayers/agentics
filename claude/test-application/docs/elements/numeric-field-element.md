## numeric-field-element.md

This file provides guidance to Claude Code (claude.ai/code) for the Numeric Input/Output Field element of the user interface.

## Numeric Input/Output Field

**Order**: 2
**Type**: Numeric
**Placeholder**: Value in Ohms

## Unfocused Actions

When the field is unselected (does **NOT** have focus):

1. The user can select colors for each color band as detailed for the **Visual Representation**
2. The application will perform the calculations detailed in the **Calculation Logic** based on the values for each color

## Focus Actions

When the field is selected (has focus):

1. The user can type in a valid numeric value of the resistance
2. The application will update the color bands as detailed in the **Visual Representation**
