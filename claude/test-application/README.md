# An Application Development Experiment Using (Anthropic) Claude #

### About ###

This code repository will hopefully by the end of things contain a complete (but small) test application developed using Claude Code.

The application as envisioned should be a simple resistor calculator. A user should be able to pick resistor band colors, and the application should display a value in ohms for the colors selected. Additionally, a user should be able to input a value in ohms, and the application should display the color bands which represent the value input.
